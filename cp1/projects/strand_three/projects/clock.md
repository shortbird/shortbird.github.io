# Clock Time

## Instructions
Write a program that takes one number as input - the number of minutes that have passed since midnight. Print the total hours and minutes that have elapsed.

## Example
**Input**
```
180 # minutes since midnight
```

**Output**
```
3 # hours
0 # minutes
```
