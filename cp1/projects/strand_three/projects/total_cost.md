# Total Cost

## Instructions
Write a program that takes three numbers. The first two numbers are the cost of one cupcake, and the third is how many cupcakes you're buying.

1. Dollars
2. Cents
3. Number of cupcakes

Print the total cost.

## Example
**Input**
```
2   # dollars
50  # cents
4   # number of cupcakes
```

**Output**
```
10  # dollars
0   # cents
```
