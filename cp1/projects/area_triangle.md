# Triangle Area

## Instructions
Write a Python program that takes two numbers as input - the length of the base and height of a right triangle. Print the area of the triangle.

## Example
**Input**
```
6
10
```

**Output**
```
30
```

## Hints
* The area of a triangle is calculated by 1/2 * base * height
